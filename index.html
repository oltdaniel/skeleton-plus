Why Skeleton-plus and not Skeleton?

Skeleton plus includes more colors, a navabr and uses a faster Font. Also Skeleton plus includes minified Versions of all needed Files. And one more Reason is the Mobile Grid System that helps you to display your layout also on smaller Devices than 550px.

Grid

Skeleton plus have the same Grid System than Skeleton. So the grid consits of 12 columns and have a max-width of 960px. Skeleton plus also includes the mobile Grid System.

Grid System

The Grid System starts mostly with an div with the row class : <div class="row">...</div>.

Input

Skeleton plus includes all skeleton button colors, so normal with a black border and a primary. The plus Version includes also danger,warning and success Button colors.

Skeleton plus also includes the normal input fields from Skeleton. But the plus Version includes a has-error and has-success Input class to change the border color.
