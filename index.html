Why Skeleton Plus and not Skeleton?

Skeleton Plus includes extra colors, a navigation bar, and a more streamlined and simple font. The new mobile grid aids in displaying content on devices smaller than 550px. Minified versions of all files are now included.

Grid

Skeleton Plus comes packed with the same grid system as Skeleton as well as a new mobile grid. This grid consists of 12 columns and has a max-width of 960px. The column count can be changed by your own in the _variables.scss.


For Devices bigger than 550px use .columns-md and for Devices smaller than 550px use .columns-sm. To use columns across every Device width use just .columns

Grid System

The grid system begins with a div with the class of "row": <div class="row">...</div>.

Mobile Grid System

The mobile grid is the same as the normal grid system but adding -sm after .column or .columns will cause the grid to display the same on mobile as it does on desktop.
Problem: If you use twelve .one column-sm the last item moves to the next line

Input

The original Skeleton provided one color, primary. Skeleton Plus adds danger,warning and success button colors.

Normal input fields have been imported from the original Skeleton. Skeleton Plus adds .has-error and .has-success input classes to change the border color.
